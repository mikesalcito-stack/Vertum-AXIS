## MATERIAL CREATION GUIDELINES

### Performance Considerations
- Use Material Instances for variations
- Optimize texture sizes (1024x1024 max for most)
- Use texture compression (BC7 for color, BC5 for normal)
- Minimize material complexity for mobile

### Visual Quality
- PBR workflow for realistic lighting
- Proper metallic/roughness values
- Consistent color palette
- Dynamic effects for interactivity

### Animation Integration
- Material parameters for dynamic effects
- Particle system integration
- Game state responsiveness
- Smooth transitions between states
